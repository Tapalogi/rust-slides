##  Estructuras de control

### if

```
let number = 6;

if number % 4 == 0 {
    println!("Numero es divisible por 4");
} else if number % 3 == 0 {
    println!("Numero es divisible por 3");
} else if number % 2 == 0 {
    println!("Numero es divisible por 2");
} else {
    println!("Numero no divisible por 4, 3, o 2");
}
```
